# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2025-12-11

### Added
- Initial release of 1NF Twitch Viewer Bot
- Custom ASCII banner with 1NF branding
- Automated proxy server integration (7 supported services)
- Selenium-based browser automation
- Automatic uBlock Origin extension download and installation
- Multiple proxy server selection (Blockaway, CroxyProxy variants)
- Interactive command-line interface with colored output
- Batch viewer sending capability
- Session management (add more sessions or exit)
- Automatic stream quality reduction to 160p
- Consent popup handling
- Success/failure statistics tracking
- Error handling and recovery mechanisms
- Windows batch files for easy installation and execution
- Comprehensive README documentation
- Requirements.txt for dependency management
- .gitignore for proper version control

### Features
- **Proxy Integration**: Seamless connection through multiple proxy services
- **Ad Blocking**: Built-in uBlock Origin for cleaner browsing
- **Multi-Session Support**: Run multiple viewer sessions without restart
- **Quality Optimization**: Automatic quality reduction for efficiency
- **User-Friendly**: Simple CLI with clear instructions

### Technical
- Python 3.x compatibility
- Selenium WebDriver integration
- Chrome browser automation
- Base64 encoding for extension handling
- HTTP requests for extension downloads
- File system operations for extension management

- Installation instructions
